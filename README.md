# ⚡ QRSendFile
QR 隔空传文件 — 利用二维码实现完全离线的文件传输工具

## 📤 发送文件
- 📁 &zwnj;**上传文件**&zwnj; — 支持点击上传或拖拽，最大 500MB
- 🧩 &zwnj;**智能分块**&zwnj; — 文件按指定字节数（默认 1500）自动切分为多个数据块，每个数据块编码为一个二维码
- 🗜️ &zwnj;**压缩传输**&zwnj; — 默认开启 pako deflate 压缩，大幅减少二维码数量。上传后即时估算压缩率和预计分块数，可随时关闭
- 🛡️ &zwnj;**纠错可选**&zwnj; — 支持 L / M / Q / H 四级纠错，平衡容量与容错
- 🔢 &zwnj;**索引标注**&zwnj; — 每个二维码中央显示 3/5 格式的序号标识
- 🔍 &zwnj;**预览模式**&zwnj; — 放大展示单个二维码便于扫描，支持自动切换（3 秒间隔），左右箭头/键盘方向键/空格键控制，点击缩略图直接进入
- 💾 &zwnj;**下载导出**&zwnj; — 支持单个下载、批量下载全部二维码 PNG 图片

## 📥 接收文件
- 📷 &zwnj;**摄像头扫描**&zwnj; — 调用后置摄像头持续扫描，自动识别 QR 码数据
- 🔭 &zwnj;**调焦控制**&zwnj; — 滑块 / +/- 按钮调节摄像头变焦，不支持变焦的设备自动隐藏
- 📊 &zwnj;**进度可视化**&zwnj; — 进度条 + 编号方块矩阵，已接收块绿色高亮，缺失块一目了然
- 🚫 &zwnj;**自动去重**&zwnj; — 同一二维码 2 秒内不重复处理
- 🖼️ &zwnj;**手动补扫**&zwnj; — 支持上传二维码截图批量解码
- 🔄 &zwnj;**自动还原**&zwnj; — 全部接收后拼接数据、自动解压（如发送时压缩过），还原原始文件并提供下载

## 🛠️ 技术特点
- 💻 &zwnj;**纯前端实现**&zwnj;，无需服务器，浏览器直接打开即可使用
- 🔒 &zwnj;**数据通过 QR 码完全离线传输**&zwnj;，不经过任何网络
- 🔀 &zwnj;**发送端与接收端可在不同设备上运行**&zwnj;（如电脑生成、手机扫描）

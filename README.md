# QRSendFile
QR 隔空传文件 — 利用二维码实现完全离线的文件传输工具

## 发送文件
    上传文件 — 支持点击上传或拖拽，最大 500MB
    智能分块 — 文件按指定字节数（默认 1500）自动切分为多个数据块，每个数据块编码为一个二维码
    压缩传输 — 默认开启 pako deflate 压缩，大幅减少二维码数量。上传后即时估算压缩率和预计分块数，可随时关闭
    纠错可选 — 支持 L / M / Q / H 四级纠错，平衡容量与容错
    索引标注 — 每个二维码中央显示 3/5 格式的序号标识
    预览模式 — 放大展示单个二维码便于扫描，支持自动切换（3 秒间隔），左右箭头/键盘方向键/空格键控制，点击缩略图直接进入
    下载导出 — 支持单个下载、批量下载全部二维码 PNG 图片

## 接收文件
    摄像头扫描 — 调用后置摄像头持续扫描，自动识别 QR 码数据
    调焦控制 — 滑块 / +/- 按钮调节摄像头变焦，不支持变焦的设备自动隐藏
    进度可视化 — 进度条 + 编号方块矩阵，已接收块绿色高亮，缺失块一目了然
    自动去重 — 同一二维码 2 秒内不重复处理
    手动补扫 — 支持上传二维码截图批量解码
    自动还原 — 全部接收后拼接数据、自动解压（如发送时压缩过），还原原始文件并提供下载

## 技术特点
    纯前端实现，无需服务器，浏览器直接打开即可使用
    数据通过 QR 码完全离线传输，不经过任何网络
    发送端与接收端可在不同设备上运行（如电脑生成、手机扫描）
